Game bernama "Tebak Kata Berantai"
game sederhana di mana pemain harus menyambung kata (misal: BUKU -> ULAR -> RUSA).

Konsep game:
-multiplayer 2 pemain
-tema kata hewan
-pemain harus menyambung huruf akhir
-ada skor
-ada hint
-ada undo
-ada nyawa (5 nyawa)

Algoritma game:

1. login pemain
2. mulai permainan
3. player 1 input katawa
4. cek validasi kata
   Benar -> lanjut
   Salah -> kurangi nyawa
5. player 2 input kata
6. cek validasi:
   Benar -> lanjut
   Salah -> kurangi nyawa
7. sampai nyawa salah satu pemain habis
8. permainan selesai
9. leaderboard
